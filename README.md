# NetRouteX – Multiple Access Simulator

## 📌 Overview
NetRouteX is a web-based simulator designed to demonstrate **FDMA, TDMA, and CDMA** concepts through interactive visualization and numerical analysis.

## 🚀 Features
- FDMA, TDMA, CDMA simulation
- Visual representation of resource allocation
- Step-by-step algorithm explanation
- User input-based computation
- Educational "Learn" section
- Real-time output display

## 🧠 Concepts Covered
- Frequency Division Multiple Access (FDMA)
- Time Division Multiple Access (TDMA)
- Code Division Multiple Access (CDMA)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📊 How It Works
1. Enter number of users, bandwidth, and time
2. Select technique (FDMA / TDMA / CDMA)
3. Click **Simulate**
4. View allocation and results

## 🎓 Educational Purpose
This project is developed as part of **Computer Networks Lab (BCSE308L)** to help students understand multiple access techniques through visualization.
